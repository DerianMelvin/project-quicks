## Getting Started

### 1. Requirements

- [Node.js](https://nodejs.org/en)

### 2. Running locally

1. Clone the repository:

```bash
git clone https://github.com/DerianMelvin/project-quicks.git
```

2. Open the project, then install all dependencies:

```bash
npm i
# or
pnpm i
```

3. Run project in development:

```bash
npm run dev
# or
pnpm dev
```
